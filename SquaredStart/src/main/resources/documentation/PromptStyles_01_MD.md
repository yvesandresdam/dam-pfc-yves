# PROMPT MAESTRO  
## Guía de Estilo para Implementación y Mejora de la Web

> Documento rector para cualquier mejora, refactorización o nueva sección desarrollada en la web.  
> Su cumplimiento debe ser estricto para garantizar coherencia visual, conceptual y funcional.

---

# 1. Objetivo General

Crear una experiencia **premium, moderna y emocional**, inspirada en el Mediterráneo, la Costa Blanca y la dualidad entre mar y montaña.

El resultado debe transmitir:

- Elegancia natural  
- Turismo de calidad  
- Tecnología amigable  
- Sensación de calma  

---

# 2. Identidad Visual

## Estética General

- Minimalista y sofisticada  
- Mediterránea y luminosa  
- Naturalista con toques tecnológicos  
- Moderna pero cálida  
- Orgánica, limpia y respirable  

## Debe evitarse

- Saturación visual  
- Colores agresivos  
- Sombras pesadas  
- Bordes duros  
- Diseños rígidos o excesivamente corporativos  

**El diseño debe sentirse abierto, relajado y premium.**

---

# 3. Paleta de Colores

## Colores Principales

| Uso | Color | Código |
|---|---|---|
| Navegación, títulos, botones | Azul Mediterráneo | `#005596` |
| CTAs y experiencias destacadas | Naranja Tierra | `#E65100` |
| Acentos visuales | Turquesa Costa | `#60D7E7` |

## Colores Neutros

- Fondo principal → `#F9F9F8`
- Blanco puro → `#FFFFFF`
- Gris humo → `#F3F4F3`
- Texto principal → `#191C1C`
- Texto secundario → `#414750`
- Bordes suaves → `#C1C7D2`

## Reglas

- Priorizar fondos claros  
- Usar azul como color dominante  
- Reservar naranja para acciones clave  
- Turquesa solo como apoyo visual  
- Mantener contraste elegante y legible  

---

# 4. Tipografía

## Fuentes

- **Headings:** Montserrat  
- **Body:** Plus Jakarta Sans  

## Jerarquía

- **H1:** 40px / Bold  
- **H2:** 32px / SemiBold  
- **H3:** 24px / SemiBold  
- **Body Large:** 18px  
- **Body Medium:** 16px  
- **Labels:** 12px / Uppercase / Bold  

## Reglas

- Titulares amplios y elegantes  
- Evitar bloques densos  
- Favorecer lectura cómoda  
- Mantener ritmo visual consistente  

---

# 5. Iconografía

## Sistema Base

- Material Symbols Outlined  
- Peso recomendado: `wght 300`

## Reglas

- Iconos simples y limpios  
- Tamaños amplios  
- Integrados en tarjetas, botones y navegación  
- Coherencia temática: naturaleza, costa, montaña, gastronomía  

---

# 6. Componentes y Formas

## Bordes

- Muy redondeados  
- Radios entre `24px` y `48px`

## Tarjetas

- Fondos blancos o neutros  
- Sombras suaves  
- Espaciado generoso  

## Reglas

- Evitar cajas rígidas  
- Sensación táctil y amable  
- Priorizar suavidad  

---

# 7. Espaciado y Layout

## Sistema Base

- Unidad: `8px`

## Escalas

- Small → `8px`
- Medium → `16px`
- Large → `32px`
- Container padding → `24px`

## Reglas

- Mucho espacio en blanco  
- Secciones amplias  
- Layout respirable  
- Evitar saturación horizontal  

---

# 8. Botones y CTAs

## Botón Primario

- Fondo azul o naranja  
- Texto blanco  
- Bordes completamente redondeados  
- Sombra suave  

## Botón Secundario

- Fondo blanco  
- Borde ligero  
- Texto azul  

## Interacción

- Hover suave  
- Active con ligera escala  
- Transiciones fluidas  

---

# 9. Experiencia de Usuario

La web debe transmitir:

- Descubrimiento  
- Confianza  
- Deseo de explorar  
- Exclusividad accesible  

## Animaciones

- Suaves  
- Elegantes  
- Naturales  
- Nunca bruscas  

---

# 10. Principios de Implementación

- Reutilizar esta paleta  
- Conservar la jerarquía tipográfica  
- Respetar radios y espaciados  
- Mantener coherencia visual absoluta  

## Regla de Conservación del Documento

**No se añadirá ningún contenido nuevo a este documento, ni se ampliarán secciones existentes, salvo que el usuario lo indique de forma expresa y directa.**

Esto incluye:

- Nuevas reglas  
- Nuevas categorías  
- Ejemplos adicionales  
- Explicaciones no solicitadas  

El objetivo es preservar la integridad del documento como marco estable y controlado.

---

# 11. Filosofía Final

Diseñar como si la web fuera una experiencia boutique de turismo mediterráneo:

**tecnológica, inspiradora y emocional.**

> No construir solo interfaces.  
> Construir atmósferas.
